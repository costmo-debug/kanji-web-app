# かんじサポート

子どもの宿題や自主学習で使うための、漢字学習Webアプリ試作品です。

## 対応範囲

- 小1から小6までの漢字
- 文章や言葉から漢字候補を探す
- 漢字一覧から探す
- お手本と書き順ガイドを表示する
- 説明、読み、例文、熟語を表示する
- クイズで復習する

## GitHub Pages での公開

このアプリは静的ファイルだけで動きます。

- エントリーポイントは `index.html`
- サーバー側の処理は不要
- ビルド作業は不要
- npm / node_modules は不要
- HTML / CSS / JavaScript だけで構成

GitHub Pages では、リポジトリの Pages 設定で公開元を `main` ブランチのルート、またはこのフォルダに合わせて指定してください。

## ローカルで試す

`index.html` をブラウザで開くと、そのまま動きます。

音声入力やホーム画面追加など、ブラウザの制限を受ける機能は、GitHub Pages などの HTTPS 上で確認するのがおすすめです。

## 書き順データについて

書き順ガイドには KanjiVG のデータを読み込んでいます。

- KanjiVG: https://github.com/KanjiVG/kanjivg
- License: Creative Commons Attribution-Share Alike 3.0

## 注意

漢字の意味・例文・熟語は試作品用のデータです。学習用として使いやすくするため、今後さらに確認・調整していきます。
